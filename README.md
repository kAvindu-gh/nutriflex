NutriFlex — Intelligent Nutrition & Fitness Ecosystem
Team Nutrition Navigators | SDGP Group SE-16

NutriFlex is a premium, data-driven mobile application designed to bridge the gap between fitness goals and nutritional reality. Developed for health-conscious individuals, the platform provides a sleek, animated environment to manage dietary intake, calculate health metrics, and streamline meal preparation.

🏛️ Project Vision
NutriFlex isn't just a calorie counter; it's a modular ecosystem built for performance and maintainability.

Premium Visual Identity: A high-contrast black-and-green aesthetic paired with smooth animations.

Modular Scalability: A clean separation between the Flutter frontend and the FastAPI backend.

Intelligent Guidance: Health metrics that adapt to specific user goals and activity levels.

🛠️ System Architecture
The system is built on a Decoupled Architecture, ensuring that the user experience is fluid while the backend remains robust and secure.

The Frontend (Client Side)
Framework: Flutter.

Focus: Real-world fitness needs with interactive food cards and "+" / "–" ingredient controls.

Key Flow: Animated recipe transitions and dynamic meal-prep dashboards.

The Backend (Server Side)
Framework: FastAPI (Python).

Security: JWT-based authentication with Bcrypt password hashing.

Pattern: Router-Service-Schema design for clean business logic separation.

🌟 Core Features
Intelligent Meal Prep: Build balanced meals with real-time calorie tracking.

Dynamic BMI Engine: Advanced calculations considering health conditions and fitness goals.

Recipe Vault: High-fidelity recipe pages with step-by-step preparation guides.

User Management: Secure profile management with encrypted authentication.

Optional Integration: Ready-made hooks for shopping and delivery workflows.
